📊 Data Analytics Dashboard

An interactive data analytics dashboard built with Python and Streamlit that allows users to explore multiple datasets through dynamic visualizations and insights.

The application provides a centralized platform for analyzing different real-world datasets including retail sales, customer segmentation, housing prices, and wine quality.

🔗 Live Application:
https://data-analytics-first.streamlit.app/

🚀 Features

Interactive dashboard built with Streamlit

Multiple datasets analyzed in one application

Dynamic Plotly visualizations

Clean dashboard layout with sidebar navigation

Dataset previews and exploratory data analysis

Interactive charts for deeper insight into data

📂 Projects Included
📈 Retail Sales Analysis

Analyzes retail transaction data to identify patterns in sales performance, category distribution, and trends.

👥 Customer Segmentation

Explores customer demographics and purchasing behavior to identify distinct customer groups.

🏡 House Price Analysis

Investigates housing market data to understand pricing distribution and property characteristics.

🍷 Wine Quality Analysis

Analyzes chemical properties of wine to explore how different attributes affect quality.
